# 📝 Task Tracker

A simple and minimal Task Tracker built using **HTML**, **CSS**, and **JavaScript**.

## Features

✅ Add new tasks  
✅ Mark tasks as completed / uncompleted  
✅ Delete tasks  
✅ Persist tasks using `localStorage`  
✅ Responsive and clean UI
